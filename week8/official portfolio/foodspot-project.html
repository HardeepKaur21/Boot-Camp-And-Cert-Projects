Project Breakdown

The FoodSpot website is a dynamic restaurant landing page built with HTML, CSS, and JavaScript, featuring a responsive layout and interactive elements.

Below are code snippets showcasing key sections of the project. These provide a peek into the implementation of major features rather than the full code structure. Each snippet comes with a brief explanation for those interested in understanding how specific functionalities were built.

🔹 Header & Navigation

This code creates a responsive navigation header for the FoodSpot website. The HTML includes a logo (.logo), a hamburger menu (.menu-toggle) for mobile screens, and a navigation list (ul) with links. The CSS styles the logo with bold white text and a red accent (span), while the menu toggle is hidden by default (display: none;), becoming interactive via JavaScript (toggleMenu()). This setup ensures a clean and adaptable navigation system.

🔹 Hero Section & Call to Action

These code snippets define a hero banner section with a full-screen background image and a text overlay. The HTML structures the banner with a title (h2), description (p), and a call-to-action button (a). The CSS sets a linear gradient overlay on top of the background image for better contrast. Flexbox ensures the content is centered. This layout provides a visually appealing and engaging introduction to the website.

🔹 About

The HTML includes a title (h3), descriptive paragraphs (p), and an image (img), all wrapped inside a flex container (.about-div). The CSS applies flexbox to position the text (.about-content) and image side by side, each taking up 48% of the width, leaving 4% of the space for the gap. The image uses object-fit: cover; to ensure it scales properly without distortion, creating a balanced and visually appealing layout.

🔹 Menu & Featured Dishes

This code defines a menu section that displays food items in a structured layout. The HTML includes a title (h3), description (p), and multiple food boxes (.food-box), each containing an image and a food name (h4). The CSS applies flexbox to ensure a responsive and centered layout. Each food box is styled with a white border and a box shadow (rgba(0, 0, 0, 0.5)), creating a clean and visually appealing design.

🔹 Experts

The HTML includes a title (h3), description (p), and multiple expert profiles (.expert-box), each containing an image and a name (h4). The CSS uses flexbox to center the content and ensure responsiveness. The section is fully wrapped (flex-wrap: wrap;) so that profiles adjust dynamically on different screen sizes, creating a well-organized and visually balanced layout.

🔹 Testimonials & Customer Reviews

This code defines a Testimonials section that displays customer reviews in a structured and visually appealing layout. The HTML includes a title (h3), description (p), and multiple customer testimonials (.customer), each containing an image and feedback. The CSS applies a background image with a dark overlay to enhance readability. Flexbox ensures the testimonials are well-aligned and responsive. This setup provides a professional and engaging way to showcase customer feedback.

🔹 Footer

The HTML contains a credit message (p) and social media links (a with SVG icons) for platforms like LinkedIn and CodePen. The CSS applies flexbox to center the content and gives the footer a bold red background (background-color: red;) for emphasis. The text is centered, ensuring a clean and structured footer design.
