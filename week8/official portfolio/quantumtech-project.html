Project Breakdown

The QuantumTech website is designed to show a different layout for a website. It is designed to be responsive. Header and footer are always visible while the content can be viewed with the help of scroll bars.

Below are code snippets showcasing key sections of the project. These provide a peek into the implementation of major features rather than the full code structure. Each snippet comes with a brief explanation for those interested in understanding how specific functionalities were built.

🔹 Header & Navigation

This code creates a responsive navigation system using HTML, CSS, and JavaScript. The hamburger menu (.burger) is hidden by default and appears on smaller screens (max-width: 600px), allowing users to toggle the navigation. Flexbox is used to align the menu items, while media queries adjust the layout for mobile devices. This approach ensures a clean, user-friendly navigation that adapts seamlessly across different screen sizes.

🔹 Sidebar & About Section

This code defines a sidebar section using HTML and CSS, designed to provide additional information and article links. The sidebar is styled using flexbox (flex: 1) to ensure it adapts within its parent container while maintaining a fixed height (93%) with scrollable content (overflow-y: auto). The HTML structure includes an "About Us" section, an image, a brief description, and a "More Articles" section with images representing additional content. This setup ensures a clean and structured layout, keeping supplementary content accessible without cluttering the main page.

🔹 Articles & Blog Layout

This code defines an articles section that takes three-fourths (flex: 3) of the layout, with the sidebar occupying the remaining one-fourth (flex: 1). The article and time elements are used for semantic clarity, improving accessibility and SEO. The CSS utilizes flexbox (display: flex; flex-direction: column;) to create a structured, scrollable layout (overflow-y: auto). Each article is styled with padding and borders (border: 1px solid #ccc;), ensuring a clean and readable design.

🔹 Footer

This code defines a simple, centered footer with a dark background (#222) and white text (#fff) for contrast. The HTML includes a call-to-action encouraging users to contribute articles, a contact link, and a tribute message. The CSS ensures the footer is visually distinct with padding (2vh 0) for spacing and centered alignment (text-align: center). The tribute acknowledges Arnold Santos, a fellow bootcamp classmate, who inspired the transformation of this layout into a tech news website.
