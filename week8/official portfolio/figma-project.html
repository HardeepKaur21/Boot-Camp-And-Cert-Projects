Project Breakdown

This project involved translating a Figma landing page into a pixel perfect representation of it in a website.

Below are code snippets showcasing key sections of the project. These provide a peek into the implementation of major features rather than the full code structure. Each snippet comes with a brief explanation for those interested in understanding how specific functionalities were built.

🔹 Header & Navigation

This code defines a navigation header with a logo, a menu, and a "Contact Us" button. The CSS styles the navigation bar with flexbox for alignment, rounded buttons, and spacing between items. The .active class highlights the current page, while the .btn class styles the contact button with a distinct background. The nav ul ensures the menu items are displayed in a row with equal spacing. 🔹 Hero Section

This code defines a hero section with a title, a short description, and a call-to-action button. The .intro-p styles the introductory text with a blurred background and rounded edges, making it visually distinct. The h1 is styled for bold typography, ensuring a strong impact, while .subtitle provides additional context. The "Buy Now" button encourages user interaction, and the "Spots Available" message adds urgency.

🔹 Image Gallery

The code snippet below defines an image gallery section displaying project images with labels. The .image class applies a gradient background, rounded corners, and a shadow for a modern look. Images inside .left and .right have adjusted padding and different border radii to create a structured layout. The .image-label p styles the descriptions with a clean, readable font to enhance presentation.

🔹 Footer

This code defines a footer section displaying a star rating and brand logos for credibility. The p is styled for readability, ensuring the rating stands out. The .stars and .logos images provide visual context, but a better approach would be to download individual logos from Figma instead of using a single container image. This would improve flexibility, allowing better responsiveness and optimization, such as adjusting spacing, hover effects, or dynamically loading different logos based on screen size.
